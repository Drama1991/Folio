## 设计决策

| 项 | 决定 | 理由 |
|---|---|---|
| Header | ☰ + Folio logo + 🔔 + 🔎，56px | 去掉桌面 SyncTicker 节省空间；🔎 全屏覆盖搜索（避免 220px 浮层问题） |
| Now Reading | 卡片化、占首屏一半 | 高频回访点，"我现在在看什么"是 Home 的灵魂 |
| 最近活动 | 时间分组（"3 天前 / 5 天前 / 一周前"） | 桌面端是卡片，移动端用紧凑行节省垂直空间 |
| 行点击 | 整行可点，→ 进入对应条目 Detail | 触控目标天然 ≥ 44px 高 |
| FAB | ⊕ 52px，距底 88px、距右 16px | 拇指右下圈；避开 iOS home indicator 区 |
| 滚动 | header sticky，FAB 也 sticky | Now Reading 滑出后 Mark 依然触手可及 |

## 交互流

- 拉到底 → 自动追加更多活动（无分页按钮）
- 下拉刷新 → 同步 NeoDB（沿用桌面端的 SyncTicker 逻辑，但视觉退化为 loading 条）
- 长按某条活动 → 弹底部 sheet，给"撤销 / 编辑 / 分享"
